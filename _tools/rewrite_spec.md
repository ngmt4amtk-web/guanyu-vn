# 關羽伝VN 本文 全面再生成 仕様書（報告調・復元エンジン版）

## 0. なぜ作り直すか（旧版の失敗）
旧版は「声の規律」で説明を削ぎ落とし、関羽一人称の測定的内面だけで通した。結果＝**いつ/どこ/誰/どういう状況/どれだけ経ったが読者に伝わらず、あらすじの羅列になり入ってこない**。これは「古典→スッと入る変換エンジン」が名指しする失敗＝**復元(RESTORE)を省いた**こと。今回は復元を正面に据える。

## 1. 新しい声（最重要・全幕共通）
**報告調の明晰な語り手（三人称・史官的）が地の文を担い、毎場面で復元する。** 台詞(say)は正史/裴注に実在する発話。関羽の内面は stated/entailed に限り軽く触れ、inferred(推論)は断定せず文脈・状況として示す（捏造回避）。立ち絵/CGが情緒を担うので、地の文は「分かること」に全力を注ぐ。

### 復元の4種（各場面で必ず埋める＝この仕様の核）
1. **指示**：人物は初出で必ず紹介（字↔本名、誰の何という立場か）。「平」→「子の関平」。ゼロ照応の主語を名指し。
2. **接地**：官職→何をする役か一言（偏将軍＝雑号より上の将軍位／盪寇将軍・襄陽太守／前将軍・仮節鉞＝独断で軍を動かし軍法を執る最高権限）。地名→どこか（白馬＝黄河沿いの渡し／樊城＝漢水を挟んで襄陽の対岸／麦城・臨沮＝荊州の山間）。度量衡→体感（七軍＝三万／二三百里）。
3. **因果・意図**：なぜそうなったかを接続詞で結ぶ。「だから」「ゆえに」「〜のために」「〜なのに」。出来事を並べず、原因→結果で繋ぐ。
4. **構造足場**：①**絶対時**（建安五年＝西暦200年、二十四年＝219年、季節）を各幕頭に置く。②**経過時間**（前の幕から何年/何ヶ月経ったか「それから八年後」等）。③**場面転換の明示**。④**幕間に起きた正史の接続事件を要約説明**（赤壁・長坂・夏口の合流、徐晃の救援 等）。

### 確信度（捏造の安全弁）
- stated＝原文に明記→自由に訳す。
- entailed＝文法・文脈で復元可能→黙って補ってよい（解凍であって発明でない）。
- inferred＝推論された内面/動機→**地の文で事実として断定しない**。「〜だったのだろう」か、状況描写で示唆に留める。

### 禁止（詩人化ドリフト＝旧版の病）
体言止めの連打／対句／三点リーダや短文での「溜めのキメ」／3連打の様式化／アフォリズム決め台詞／抽象名詞（孤高・天命・運命）を主語にした感慨。**守るのは語彙でなくリズム＝原文の一息の運動を、勿体ぶらず素直に流す。** 硬質さはリズムでなく情報密度で出す。

### 考証（時代錯誤・演義の排除）
排除確定：青龍偃月刀・赤兎の固有名／桃園結義・義兄弟の盟誓／五虎大将（呼称）／温酒斬華雄／三英戦呂布／土山約三事・保護二嫂／過五関斬六将・千里走単騎／単刀赴会の主導演出／華容道／華佗執刀・刮骨の囲碁／文醜斬殺の手柄化／能動的水攻め（堰切り）／虎女犬子の名句（正史の語は「貉子(鰂子)」）／玉泉山顕聖・周倉。関羽の得物は矛・環首刀のみ。近代器具・椅子等を出さない。

### ビート粒度
1ビート＝1〜2文（1タップで読み切れる量）。場面が変わる/時が飛ぶ/話者が変わるたびにビートを割る。アセット(bg/sprite/cg)は下記スケルトンの実在キーのみ使用（新規生成なし）。立ち絵は発話/対峙する人物のみ。語り手は立ち絵の無い人物（徐晃・司馬懿・蔣濟・呂蒙・糜芳・士仁等）を地の文で名指し説明してよい。

## 2. 人物事典（初出で紹介・以降は短く参照）
- **関羽（字＝雲長、本字＝長生）**：本伝の主人公。河東郡解県の人。長い美髯。劉備の最も古い将。
- **劉備（字＝玄徳、本作では「先主」とも）**：関羽が生涯ただ一人誓った主君。後に蜀漢の昭烈帝。
- **張飛（字＝益徳）**：関羽と並ぶ劉備の古参の将。関羽とは兄弟同然。
- **曹操（本作「曹公」）**：後漢の実権者・魏の礎。下邳で関羽を捕らえ厚遇した。
- **張遼（字＝文遠）**：曹操配下の将。元は呂布の部下。関羽と親しく、去就を探る使者となる。
- **顔良**：袁紹配下の猛将。白馬で関羽に斬られる。
- **諸葛亮（字＝孔明、「軍師」）**：劉備の軍師。成都にいて関羽と書簡を交わす。
- **馬超（字＝孟起）**：涼州の猛将。潼関で曹操を破った後に劉備へ降る。
- **于禁**：曹操配下の宿将。樊城救援の七軍を率い、水没して関羽に降伏。
- **龐徳**：于禁軍の将。降らず関羽に斬られる。
- **孫権**：江東の主。関羽の背後を襲い、結果的に関羽を滅ぼす。
- **呂蒙**：孫権配下の将。白衣の計で江陵を奪う（立ち絵なし＝関羽は対面しない／地の文で説明）。
- **関平（子の平）**：関羽の子。常に従い、臨沮で共に斬られる。
- **糜芳・士仁**：荊州の留守を預かる将（糜芳＝南郡太守・江陵／士仁＝公安）。関羽に侮られ、孫権に内応（立ち絵なし）。
- **徐晃・司馬懿(司馬宣王)・蔣濟・曹仁・劉延**：地の文で名指し説明する脇（立ち絵なし）。曹仁＝樊城を守る曹操の将。劉延＝白馬を守る東郡太守。

### act05 漢寿亭侯 ─ 報いて、去る
[正史]「初，曹公壯羽為人，而察其心神無久留之意，謂張遼曰：卿試以情問之。」「既而遼以問羽，羽歎曰：吾極知曹公待我厚，然吾受劉將軍厚恩，誓以共死，不可背之。吾終不留，吾要當立效以報曹公乃去。」「遼以羽言報曹公，曹公義之。」「及羽殺顏良，曹公知其必去，重加賞賜。」「羽盡封其所賜，拜書告辭，而奔先主於袁軍。」「左右欲追之，曹公曰：彼各為其主，勿追也。」 +裴注傅子「遼欲白太祖，恐太祖殺羽...乃歎曰：公，君父也；羽，兄弟耳。遂白之。太祖曰：事君不忘其本，天下義士也。」

### act07 髯 ─ 馬超評と諸葛亮の書

### act08 刮骨 ─ 言笑自若

### act11 貉子 ─ 孫権への罵倒と背後の謀
[正史]「司馬宣王、蔣濟以為關羽得志，孫權必不原也。可遣人勸權躡其後，許割江南以封權，則樊圍自解。曹公從之。」「先是，權遣使為子索羽女，羽罵辱其使，不許婚，權大怒。」+典略「羽圍樊，權遣使求助之...羽忿其淹遲...乃罵曰：貉子（鰂子）敢爾，如使樊城拔，吾不能滅汝邪！」「又南郡太守麋芳在江陵，將軍士仁屯公安，素皆嫌羽輕己。羽之出軍，芳、仁供給軍資，不悉相救。羽言還當治之，芳、仁咸懷懼不安。」
[復元] **報告調なので曹操陣営の謀を語り手が説明してよい（旧版の全知問題は解消）**：曹操の参謀**司馬懿・蔣濟**が「関羽の勢いに孫権も不安。孫権を誘って背後を襲わせ、江南を割いて封じると約せば、樊の囲みは自ら解ける」と進言し曹操が容れた、という外交の構図を明示。先に孫権が子の嫁にと関羽の娘を求めたが関羽が使者を罵り拒んだ経緯（典略「貉子」＝むじなの子、と格下げした罵り。虎女犬子は使わない）。さらに留守の糜芳・士仁が、補給を十分にせず関羽に「還ったら処分する」と言われ不安を抱いていた＝背後が崩れる前提を、語り手が因果として説明。

### act12 麦城 ─ 徐晃の救援、江陵陥落、軍散る
[正史]「於是權陰誘芳、仁，芳、仁使人迎權。而曹公遣徐晃救曹仁。」「羽不能克，引軍退還。權已據江陵，盡虜羽士眾妻子，羽軍遂散。」+裴注呂蒙伝「伏精兵於𦩷𦪇之中，使白衣搖櫓，作商賈服」
[復元] **省かれていた因果を全部復元**：孫権がひそかに糜芳・士仁を誘い、二人は内応して孫権を迎え入れた→一方曹操は**徐晃**を遣わして曹仁を救援した（＝関羽は樊を抜けなくなった理由）→関羽は樊を落とせず軍を退いた→だが背後では、**呂蒙が兵を商人の白衣に変装させ船に潜ませる奇計（白衣渡江）で江陵を既に奪っており**、関羽の将兵の妻子はことごとく捕らわれていた→家族を人質に取られた将兵は次々離散し、関羽の軍は崩れていった、という連鎖を順に。麦城へ追い詰められる状況。
[アセット] BG=bg_bakujo_haisou_yoru(night) / SP=guanyu_late, guanping_base
[注] 呂蒙・徐晃・糜芳・士仁は地の文で説明（立ち絵なし）。

### act13 臨沮 ─ 斬られる、そして後日譚
[正史]「權遣將逆擊羽，斬羽及子平於臨沮。」「追諡羽曰壯繆侯。」「子興嗣...弱冠為侍中、中監軍，數歲卒。子統嗣，尚公主，官至虎賁中郎將。卒，無子，以興庶子彝續封。」+裴注吳書「孫權遣將潘璋逆斷羽走路」、吳曆「權送羽首於曹公，以諸侯禮葬其屍骸」
[復元] 孫権が将**潘璋**を遣わして関羽の退路を断ち、臨沮（江陵から二三百里の山間）で関羽と子の関平を捕らえて斬った、を接地。最期は劇的に誇張せず正史の冷たい一行の重みで（語り手の声で）。続けて後日譚を史官の筆致で：諡＝壮繆侯、孫権が首級を曹操に送り諸侯の礼で葬ったこと、子の関興が嗣ぎ早世、関統、庶子の関彝へと爵が継がれたこと。玉泉山顕聖・神格化は完全排除。CGは囲みと退路断絶（斬首の瞬間は描かない）→史官の記録bgで後日譚。
