# 關羽伝VN 本文 改訂v2 設計図（会話付与・重複整理・時系列整序）

現行本文は `~/guanyu-vn/scenes.js`（window.SCENES）。各幕は §6 の指示で改訂する。**報告調の復元ナレーションは資産として保持**し、その上に会話を足し、重複と時系列を整える。

## 0. 今回やること（ユーザー指摘の反映）
1. **会話ゼロを解消**。各場面、ナレーションで状況を説明した「後」に、その描写を立ち上げる会話(say)を入れる。**ナレーションは減らさない**。
2. **会話・性格は三国無双(Dynasty Warriors)を参照しつつ、史実記録からの想像でよい**（＝Tier3＝内面/台詞の劇化を解禁。ただし出来事の骨格は正史から変えない）。
3. **重複説明をカット**。幕をまたいで同じ用語・人物・地理を説明し直さない（§3台帳の「初出」でのみ説明、以降は素の名前）。
4. **時系列を整序**（シャッフル禁止）。§4の通り張遼問答＝誓いをact03へ移し、act05の「時の前後がねじれている」式の回想を廃す。
5. 個別修正（act03の因果順／act07馬超の文）は§6で指定。

## 1. 会話の付け方（全幕共通）
- 構造＝**ナレーション（状況・場所・経過の復元）→ その場面の会話（say）→ 必要なら締めのナレーション**。会話は地の文の「直後」に置き、地の文が説明したことを人物の口・やり取りで体感させる。
- 会話は**史実の出来事の範囲内**で想像してよい（正史/裴注にある発話はそのまま活かし、無い所は人物像から自然に補う）。**新しい出来事・正史と矛盾する事実は作らない。**
- 話者には立ち絵が要る（その幕の利用可能立ち絵のみ。§6参照）。立ち絵の無い相手（使者など）は name だけで say してよい（立ち絵なし）。一人の場面は短い独白(say)可。
- **禁止**：演義のみの逸話を事実として混ぜる（桃園/五虎/青龍偃月刀/赤兎/虎女犬子/華佗 等）／近代語・時代錯誤語／詩人化（体言止め連打・対句・3連打・キメ・アフォリズム）。会話は自然な現代語で、芝居がかりすぎない。

## 2. 人物の性格（三国無双系・台詞の指針）
- **関羽**：誇り高く義に篤い。寡黙で威厳、武と忠に揺るがぬ自負。慇懃で硬質な物言い。
- **劉備**：仁徳・理想家。人を思いやり、兄弟を温かく遇する。穏やかで情のある語り。
- **張飛**：豪放・短気・直情。声が大きく粗野だが忠義。関羽を「兄者(あにじゃ)」と呼ぶ。
- **曹操**：英雄的・現実的・才を愛する。関羽を欲しがる。自信に満ち、時に芝居気。
- **張遼**：実直・律儀。忠と情の板挟みを誠実に生きる。関羽に敬意。
- **諸葛亮**：沈着・聡明。優雅で含みのある言葉。
- **于禁**：歴戦の宿将、規律。降伏は苦渋。 **龐徳**：剛直、死をもって節を守る。
- **関平**：父を敬う実直な若武者。 **使者(孫権の)**：慇懃だが尊大。

## 4. 時系列の整序（シャッフル禁止・act03〜05）
正史の因果：張遼が関羽の本心を探る→関羽が「功を立てて報いてから去る」と誓う→**白馬で顔良を斬る＝その報恩の実行**→劉備の所在を知り去る→曹操「追うな」。これを時系列順に並べる。
- **act03**：捕囚→厚遇。**末尾に張遼問答＝誓いを置く**（曹操が張遼に探らせる／関羽「劉将軍と誓って共に死ぬ身、功を立てて報いてから去る」／傅子の張遼の板挟み「公は君父、羽は兄弟」／曹操「天下の義士」）。
- **act04**：白馬で顔良を斬る。**斬殺の後に一言、「これが関羽の言った“報いてから去る”の実行だった」と地の文で結ぶ**（act05へ繋ぐ）。
- **act05**：「時の前後がねじれている」式の回想・メタ説明を**全廃**。内容＝関羽が劉備の所在（袁紹軍）を知る→賜物を封じ書を残して去る→曹操「彼各為其主、勿追也」。張遼問答はもうここに置かない。素直な時系列。

## 6. 各幕の改訂指示（現scenes.jsの当該actを読み、ナレーション保持＋会話付与＋重複削除＋下記修正でbeats再構成）
- **act01**：関羽ひとりの逃避行。会話は短い独白1つ可（北で生き直す決意など）。長い説明は保持。
- **act02**：劉備・張飛との出会い。**会話を足す**：劉備が義兵に関羽を迎える温かい言葉／張飛の豪放な一言（「兄者」と呼ぶ流れ）／関羽が身を投じる応え（桃園の盟ではなく、自然に従う）。寝同床/恩若兄弟の地の文は保持。
- **act03**：**因果順を整える**＝「劉備が流転を重ね、ついに徐州一帯を収めた（車冑を殺し徐州を奪った）→その要・下邳城を関羽に委ね太守を代行させた」を先に語り、そこから曹操東征→劉備敗走→関羽捕縛→厚遇、と進む。**関羽の再紹介（字・出自）を削除**。会話＝曹操が関羽の武と義を惜しみ将に欲しがる言葉／関羽の慇懃だが距離ある応え。**末尾に§4の張遼問答＝誓いを会話で**置く。
- **act04**：関羽の再紹介を削除。会話＝張遼と関羽の短い陣前のやり取り（敵の数／顔良の麾蓋を見つけた関羽）。顔良斬殺はCG＋地の文。**斬殺後に「これが“報いてから去る”の実行」の一文**。漢寿亭侯。得物は矛。
- **act05**：§4の通り回想メタを廃し時系列を整序。関羽再紹介削除。張遼問答は置かない。会話＝去るにあたっての関羽の一言（劉備のもとへ向かう決意）／曹操「追うな」。
- **act06**：赤壁・長坂・夏口・益州・董督荊州の復元ナレーションは保持。会話を1〜2足す：劉備が荊州統括を関羽に託す言葉、または関羽が江北を鎮める決意（短く）。関羽再紹介削除。盪寇将軍等の用語は初出なので説明可。
- **act07**：関羽再紹介不要。**馬超の変な文を修正**：「だが関羽にとって馬超は旧来の知り合いではなく…決めかねた」を、自然な一文へ（例「馬超は関羽の旧知ではない。その人となりを測りかね、関羽は成都の諸葛亮に書を送って尋ねた」）。会話＝関羽が書を読んで誇らしげに賓客へ示す一言（無双系の自負）／関平の反応も可。諸葛亮の返書は保持。
- **act08**：刮骨。医者の説明は保持。**会話を足す**＝治療の最中、関羽が平然と諸将と談笑する台詞（言笑自若を会話で体感／無双系の豪胆）、関平の気遣い。
- **act09**：前将軍・仮節鉞・樊城は初出＝説明可。会話＝漢中王の使い／関平が任を告げる／関羽が樊攻めへ向かう決意。背後（糜芳・士仁）への布石は地の文。
- **act10**：**§5の重複カット**（樊城等の再説明を削る）。会話＝于禁の降伏の苦渋の言葉／龐徳の不屈の罵り（正史の龐徳台詞を活かす）／関羽が龐徳を斬る判断の一言。水淹七軍・威震華夏の地の文は保持。
- **act11**：**§5で冒頭の樊城再説明カット**。司馬懿・蔣濟の謀／孫権の縁談と関羽の拒絶＝**会話で**（孫権の使者の慇懃尊大な口上／関羽の拒絶・「貉子」の罵り）。糜芳・士仁の不安は地の文（初出説明はここ）。
- **act12**：孫権/呂蒙/江陵の**再説明をカット**（素の名前で）。徐晃の救援＝関羽が抜けなかった理由は初出説明可。会話＝関平が離反・江陵陥落を報じる／関羽の動揺と無念（無双系の悲壮）。白衣渡江は地の文で。
- **act13**：act12の粗筋反復を削り短い接続に。会話＝麦城を脱する関羽と関平のやり取り（父子の最後／関羽の不退転）／囲まれての最期は劇的誇張せず。斬首はCGと地の文。後日譚（諡・継承）は史官の地の文。
